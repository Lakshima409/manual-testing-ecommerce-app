# manual-testing-ecommerce-app
Manual testing of an e-commerce web application including test plan, test scenarios, test cases, and bug reports to demonstrate practical QA skills and industry-standard testing practices.
Project Overview This project demonstrates manual testing of a web-based e-commerce application to ensure functionality, usability, and reliability. 
Scope of Testing - User Registration - Login Functionality - Product Search - Add to Cart - Checkout Process 
Tools Used - MS Excel - GitHub - Web Browser (Chrome) 
Artifacts Included - Test Plan - Test Scenarios - Test Cases - Bug Reports 
Key Learnings - Test case design techniques - Defect reporting and tracking - Understanding real-world QA workflows
